How the Genetic Algorithm Works

1
Start with Nature: Begin with a real algae protein sequence that already processes CO2
2
Create Population: Generate 100 variations of this protein
3
Evaluate Fitness: Test each protein for CO2 binding, stability, and catalytic power
4
Select Best: Keep the top-performing proteins for the next generation
5
Evolve: Combine features from good proteins and add small random changes
6
Repeat: Continue this process for 50 generations until optimal
